1.1. Requisiti dell'host

  • Una connessione ad Internet (solo in fase di installazione)

  • GNU Wget (solo in fase di installazione)

  • Un nome di dominio

  • Un webserver (ad esempio, Apache HTTPd)

  • Un certificato SSL valido (opzionale, ma raccomandato)

  • Docker Engine

  • Docker Compose

Suggerimento

È possibile ottenere gratuitamente un certificato SSL utilizzando Letsencrypt!
